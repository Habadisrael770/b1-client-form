# טופס הקמת לקוח

טופס הקמת לקוח מודרני עם עיצוב מקצועי ותמיכה בעברית.

## תכונות

- ✅ עיצוב מודרני וריספונסיבי
- ✅ תמיכה מלאה בעברית (RTL)
- ✅ וולידציה של שדות בזמן אמת
- ✅ תמיכה בהעלאת קבצים
- ✅ שעות חלוקה גמישות
- ✅ ניהול אזורי חלוקה
- ✅ שעות עבודה מותאמות

## שימוש

פתח את `index.html` בדפדפן כדי להשתמש בטופס.

## שליחת נתונים

הטופס משלח נתונים ל-Formspree. עדכן את ה-URL של Formspree בקוד:

```javascript
fetch("https://formspree.io/f/YOUR_FORM_ID", ...)
```
